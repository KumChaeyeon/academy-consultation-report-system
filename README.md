# Academy Consultation Report System

## 1. 프로젝트 개요

학원 상담 데이터를 활용한 월간 상담 리포트 자동 생성 시스템 구축 프로젝트.

기존 학원 운영 환경에서 상담 데이터 확인 및 통계 분석 과정의 수작업 의존성 존재. 상담 건수, 등록 전환율, 학년별 상담 현황, 학교별 상담 현황 등의 분석 과정에서 반복적인 엑셀 작업 필요성 존재.

상담 데이터 자동 처리 및 데이터 기반 의사결정 지원 목적의 데이터 파이프라인 구축 목표. 데이터 수집, 정제, 저장, 분석, 리포트 생성까지의 전 과정을 자동화하는 시스템 구현 목표.

---

## 2. 시스템 구조

전체 데이터 흐름 구조.

```text
상담 데이터(Excel)
        ↓
데이터 로딩
        ↓
데이터 정제
        ↓
PostgreSQL Staging Layer
        ↓
PostgreSQL Core Layer
        ↓
SQL 기반 데이터 분석
        ↓
월간 상담 리포트 생성
```

데이터웨어하우스 구조 적용.

* Staging Layer : 원본 데이터 저장 목적
* Core Layer : 분석용 데이터 저장 목적

데이터 적재와 분석 과정 분리 구조.

## 3. 데이터 처리 과정

### 데이터 로딩 과정

Pandas 라이브러리 활용.

상담 데이터 엑셀 파일 읽기 기능 구현.

필요 컬럼 추출 기능 구현.

* 상담일자
* 학년
* 이름
* 학교
* 상담내용
* 등록여부

불필요 컬럼 제거 과정 수행.

---

### 데이터 정제 과정

* 데이터 품질 향상 목적의 전처리 과정 수행.

#### 날짜 데이터 변환

* 문자열 형태 데이터의 datetime 형식 변환 과정 수행.

#### 개인정보 제거

* 이름 컬럼 제거 과정 수행.

* 개인정보 비식별화 목적.

#### 등록여부 정규화

* 전환 → 1 / 미전환 → 0

* 형태의 데이터 변환 수행.

* 등록 전환율 계산 목적의 정규화 과정.

## 5. 데이터 분석 기능

* SQL 기반 분석 기능 구현.

### 상담 건수 분석

* 전체 상담 건수 집계 기능.

* 상담 수요 변화 추이 분석 목적.

### 학년별 전환율 분석

### 학교별 전환율 분석
